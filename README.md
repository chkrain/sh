1. **Сделайте скрипты исполняемыми:**
chmod +x *.sh

2. **Сделайте скрипты исполняемыми:**
./system_manager.sh start

3. **Запустите систему:**
./system_manager.sh start

# Для автозапуска выполнить:
echo "@reboot /home/ainur/Desktop/sh-te/system_manager.sh start" | crontab -

system_manager.sh - Главный демон
Назначение: Управляет всеми остальными скриптами, запускает их по расписанию

Интервалы по умолчанию:

Проверка диска: 24 часа

Проверка Git: 2 часа

Напоминания: 1 час

Проверка сети: 5 минут

Управление: start|stop|status

disk_check.sh - Мониторинг дисков
Назначение: Проверяет свободное место на разделах

Пороги: Предупреждение (30%), Критический (15%)

Действия: Уведомления, автоматическая очистка при критическом уровне

Безопасность: Проверяет только системные разделы, не трогает пользовательские данные

disk_cleanup.sh - Очистка и резервное копирование
Назначение: Автоматическая очистка и резервное копирование старых проектов

Действия:

Резервное копирование проектов не использовавшихся 90+ дней

Очистка временных файлов

Предложения по удалению заархивированных проектов

Безопасность: Создает коммиты, НЕ удаляет проекты автоматически

git_check.sh - Мониторинг Git
Назначение: Поиск репозиториев с несохраненными изменениями

Проверяет: Неотслеживаемые файлы, незакоммиченные изменения, необходимость пуша

Безопасность: Только читает статус, не выполняет автоматические коммиты

network_check.sh - Мониторинг сети
Назначение: Проверка доступности хостов и качества соединения

Проверяемые хосты: 8.8.8.8, google.com, github.com

Действия: Уведомления при проблемах с сетью

break_reminder.sh - Напоминания
Назначение: Напоминания о необходимости разминки

Упражнения: Случайные упражнения для глаз, спины, общая разминка

Интервал: Каждый час
